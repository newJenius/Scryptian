Scryptian

A lightweight command bar that keeps your content local. Trigger with Ctrl + Alt, run skills from your clipboard, and let the bundled Qwen 3B model handle rewriting, fixing, translating, and more — all offline.

What's inside

Instant skills

Translate, summarize, fix code, clean JSON, convert casing — all within a single bar. Add your own skills by dropping a .py file into the skills/ folder.

Offline model

The first run downloads a 2 GB Qwen 3B GGUF model and caches it locally. Subsequent runs are instant. No cloud calls, no API keys.

System tray friendly

Launch on startup, stay out of your way, and quit anytime from the tray icon. Fully self-contained — no installers or services.

Transparent telemetry

PostHog events track only high-level actions (hotkeys, downloads) to improve reliability. No clipboard data is ever sent.

Quick start

1. Download

Grab the latest Scryptian.exe and place it anywhere. Double-click to launch — the tray icon will appear near the clock.

2. Allow autostart (optional)

Scryptian registers itself for startup the first time it runs. You can manage it later from the Windows Startup Apps settings.

3. Use the hotkey

Press Ctrl + Alt (customizable in config.py) to open the bar. Paste text, pick a skill, press Enter. Results copy to the clipboard automatically.
